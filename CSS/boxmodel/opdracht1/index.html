wat zijn de eigenschappen van block box en inline box?

block box
  1. de box zal in een nieuwe regel breaken
  2. margin,padding en border zorgen ervoor dat andere elementen van de box worden weggeduwd
  3. de box gaat de heel browser breed nemen
  4. de width en height eigenschappen zijn gerespecteerd
Inline-block box
  1. de box zal niet in een nieuwe regel breaken
  2. de width en height eigenschappen zullen niet toepassen
  3. Vertical padding, margins, and borders zijn van toepassing, maar zorgen er niet voor dat andere inline vakken van de box wegschuiven
  4. de box bezet evenveel breedte als de breedte van hun inhoud
